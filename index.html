Malayalam Type Judge


Test your typing skills in Malayalam!


“Typing feedback is triggered based on speed (Characters Per Second):
- < 2 CPS: Slow - corrective feedback
- 2-4 CPS: Moderate - neutral response
- > 4 CPS: Fast - praise delivered”






📝 How to Install:

  1. Download the generated ZIP file
  2. Unzip it to a folder
  3. Open Chrome → go to chrome://extensions/
  4. Turn on "Developer mode" (top right toggle)
  5. Click "Load unpacked" and select your unzipped folder
  6. Start typing anywhere and get judged in Malayalam! 🎭
